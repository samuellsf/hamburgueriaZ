# 🍔 HamburgueriaZ

Aplicativo Android desenvolvido em **Java** no **Android Studio** como projeto acadêmico, com o objetivo de simular o sistema de pedidos de uma hamburgueria.

---

## 📱 Sobre o Projeto

O aplicativo permite que o usuário:

- Informar nome do cliente
- Selecionar quantidade de lanches
- Escolher adicionais (bacon, queijo e onion rings)
- Visualizar resumo do pedido em tempo real
- Calcular automaticamente o valor total
- Enviar pedido por **E-mail ou WhatsApp**

---

## 🛠 Tecnologias Utilizadas

- Java
- Android Studio
- ConstraintLayout
- Intents (integração com apps externos)
- Componentes Android (TextView, EditText, CheckBox, Button)

---

## 💡 Aprendizados

Durante o desenvolvimento, foram aplicados conceitos como:

- Lógica de programação
- Programação Orientada a Objetos (POO)
- Manipulação de eventos
- Validação de dados
- Cálculo dinâmico
- Integração entre aplicações
